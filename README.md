# Active-Directory-Basics
🧠 Overview
The goal of this lab was to learn and practice foundational Active Directory (AD) administration concepts, including:
Provisioning a Windows Server VM in Azure
Installing Active Directory Domain Services (AD DS)
Promoting the server to a Domain Controller
Building a logical AD structure with Organizational Units (OUs)
Creating users
Creating security groups and assigning users

---

🛠️ Lab Environment
Cloud Provider: Azure
Server OS: Windows Server (used for Domain Controller)
Domain Services: Active Directory Domain Services (AD DS)
Tools Used:
Server Manager (GUI)
Active Directory Users and Computers (ADUC)
Remote Desktop Protocol (RDP)
This setup provided a practice environment that mirrors real-world IT infrastructure administration.
